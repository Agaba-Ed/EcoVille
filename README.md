# EcoVille
Make the world cleaner

## Installation
Clone this repository and import into **Android Studio**
```bash
git clone git@github.com/Agaba-Ed/EcoVille.git
```

## Generating signed APK
From Android Studio:
1. ***Build*** menu
2. ***Generate Signed APK...***
